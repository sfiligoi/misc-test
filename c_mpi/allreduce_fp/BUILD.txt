Example build and run instructions
==================================

Cray compiler for GPU
---------------------
Build:
cc -fopenmp -Ofast -o mpi_omp_gpu_benchmark mpi_omp_gpu_benchmark.c
Run:
srun -n4 ./mpi_omp_gpu_benchmark 1000000 2000

Cray compiler for CPU
---------------------
Build:
cc -fopenmp -Ofast -DNO_GPU -o mpi_omp_gpu_benchmark mpi_omp_gpu_benchmark.c
Run:
srun -n4 ./mpi_omp_gpu_benchmark 1000000 2000
